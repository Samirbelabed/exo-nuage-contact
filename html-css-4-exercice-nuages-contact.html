astuce pour aligner des blocs

Ce n'est pas simple d'aligner des blocs, il y a des solutions que nous verrons dans quelques jours, donc pour aller au plus rapide, concernant les deux colonnes pour les éléments de formulaires vous pouvez utiliser le css suivant.
Pour le parent qui engloble les deux colonnes :

  .flexRow {
  display:flex;
  flex-direction: row;
  }

Pour les deux colonnes enfants :

.colonne50 {
  flex-basis: 50%;
}
